# Raku Distribution Dependencies

Find the real dependencies used in a distribution, and act on them.

## Installing

Just the usual

    zef install Distribution::Dependencies

## Running

```
raku -Ilib -MDistribution::Dependencies -e 'say explore "../raku-some-distro
"' cd raku-some-distro


```
